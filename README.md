# Smart Home

A GUI that simulates controls for a smart home.

## Setup

### Pre-requisites

[Poetry](https://python-poetry.org/) needs to be installed.

### Install

Install the dependencies:

```bash
poetry install --no-root
```

### Run

Run the program:

```bash
poetry run python3 ./frontend.py
```
